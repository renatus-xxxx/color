## overview
COLOR command for msx-dos

## how to use
```
usage:
  COLOR <text color> <background color> <border color>
  COLOR /env (auto background color set by ENV.III on MSX0)
```
## dependency
- [libiot](https://github.com/renatus-xxxx/libiot)
